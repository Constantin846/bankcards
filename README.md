# Bankcards

### Описание

Данный проект создан в учебных целях.

Приложение хранит список клиентов и их банковские карты.

Функционал приложения:

* Возможности администратора:
    * Регистрация / обновление клиентов
    * Регистрация / блокировка / активация / удаление банковских карт
    * Получение краткой информации о всех банковских картах
* Возможности клиента:
    * Получение краткой и полной информации о собственных банковских картах
    * Перевод средств между своими банковскими картами
    * Регистрация запроса на блокировку собственной банковской карты

### Запуск проекта

**<span style="color:orange">Внимание!</span>** Для запуска проекта необходимы git, maven и docker

* Склонируйте репозиторий: git clone https://github.com/Constantin846/bankcards.git
* Запустите docker до создания исполняемого jar
* Откройте PowerShell или Terminal или CMD
* Перейдите в директорию проекта: cd {your_path}/bankcards
* Создайте исполняемый jar: mvn package
* Создайте и запустите docker контейнеры: docker-compose -f docker-compose.yml up

### API

Ознакомиться с API можно после запуска приложения по ссылке http://localhost:8080/swagger-ui/index.html
